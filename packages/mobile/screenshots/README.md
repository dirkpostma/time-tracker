# App Store Screenshots

Place screenshots in the appropriate device folder.

## Required Screenshots

### 6.9/ (iPhone 15 Pro Max - 1320×2868)

| File | Description |
|------|-------------|
| `01_timer.png` | Timer screen with client/project selected |
| `02_timer_running.png` | Timer actively running |
| `03_history.png` | History screen with multiple entries |
| `04_settings.png` | Settings screen showing theme options |
| `05_themes.png` | Multiple themes side-by-side or theme picker |

### 6.7/ (iPhone 14 Pro Max - 1290×2796)

Same screenshots as 6.9/, optional but recommended.

### ipad/ (iPad Pro 12.9" - 2048×2732)

Same screenshots, required since app supports tablet.

## Capture Instructions

1. Run app in appropriate simulator
2. Set up realistic demo data
3. Press Cmd+S to capture
4. Move to this folder and rename appropriately

See APP_STORE_SUBMISSION.md for detailed instructions.
